# React + Vite

Name of my Project : DigiTools Platform
A little Description : DigiTools Platform is a modern web application that provides a collection of powerful digital tools designed for creators, professionals, and businesses. It helps users boost productivity and simplify their workflow through an intuitive and responsive interface.

Technology used :
React.js
Tailwind CSS, DaisyUI
JavaScript (ES6+)
React-Toastify (NPM Package)
JSON (for product data)
Github
Netlify - Deployment

3 Features about my project :

   a. Add to Cart System : Users can easily select and add tools. Dynamic cart management with add/remove functionality

   b. Toast Notifications : Provides instant feedback for user actions. Displays attractive notifications when items are added or removed.

   c. Interactive UI & Responsive Design : Clean and user-friendly interface. Fully responsive across mobile, tablet, and desktop devices
